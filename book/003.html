# d3教程

# 读取表格数据

# csv tsv dsv

d3提供了三个不同的函数来拉入表格数据：d3.csv（）、d3.tsv（）和d3.dsv（）。它们之间的唯一区别是d3.csv（）是为逗号构建的分隔文件，d3.tsv（）是为制表符分隔的文件构建的，d3.dsv（）允许 声明分隔符。

# d3核心流程

加载数据--->格式化数据--->进一步修改数据(数据分类等)--->测量和排序--->数据绑定

d3加载的五种文件的方法d3.text() , d3.xml() , d3.json() , d3.csv() , d3.html()

这些方法都是异步的，需要避免数据加载中后面的操作函数已执行。

d3.csv("somefiles.csv", function(data) {doSomethingWithData(data)});

或者使用promise.

function interpolate(t){
	return a*(1-t)+b*t

# 中间帧

tween函数式一个工厂函数，用来构造执行中间帧计算的最终函数 ；

.attrTween('d', function(d, i) {
	let interpolate = d3.interpolate(d.startAngle, d.endAngle);
	return function(t) {
		d.endAngle = interpolate(t);
		return arc(d);
	}
})

        return function(t){ // <-D
            return interpolate(t);
        };

安装mongodb后，配置环境变量.

找到安装的mongodb下的bin目录，配置在path里。

D:\all\mongo\bin

# 启动数据库 mongodb

  • mongod
exception in initAndListen: NonExistentPath: Data directory C:\data\db\ not found., terminating

可以按照报错提示在c盘下新建一个data/db目录，重启即可

  • 用管理员cmd net start mongodb或者mongod(要每次都指明data\db)
 如:mongod --dbpath e:/MongoDB/data/db

提示

如果发生无法连接code:100，可能是mongodb上次关闭有问题，需要把data下的.lock文件删除后再试。

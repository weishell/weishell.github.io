# react

使用 create-react-app 快速构建 React 开发环境

cnpm install -g create-react-app
create-react-app my-app
cd my-app/
npm start

my-app/
  README.md
  node_modules/
  package.json
  .gitignore
  public/
    favicon.ico
    index.html
    manifest.json
  src/
    App.css
    App.js
    App.test.js
    index.css
    index.js
    logo.svg

manifest.json 指定了开始页面 index.html，一切的开始都从这里开始，所以这个是代码执行的源头。

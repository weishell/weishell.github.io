# d3-interpolate

给定函数f(x)在x0，x1...，xn出的值，现有x'，其值在上述取值点之间，那么求f(x')近似值的过程就叫作插值

cnpm install d3-interpolate --save

var it = d3.interpolateNumber(10,100)
		console.log(it(0.5))
		//<====>
		function it(t){
			return a*(1-t)+b*t
			//10(1-t)+100t
		}

a代表值域的起点，b代表值域的终点，传递的参数0~1之间，它指明返回值与a的距离~

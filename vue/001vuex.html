# vuex简单使用

每一个 Vuex 应用的核心就是 store（仓库）。“store”基本上就是一个容器，它包含着你的应用中大部分的状态 (state)。 当页面刷新的时候，vuex的数据会丢失(重置)，所以，有的数据刷新可以丢失，有的不可以，如果不可以的话，需要配合localstorage来存储。

# 安装vuex

npm install vuex --save

理解

vuex理解 官方解释：vuex是一个专为vue.js应用程序开发的 状态管理模式。它采用集中式存储管理应用的所有的状态，并以相应的规则保证状态以一种可预测的方式发生变化。 可以理解为是一个 全局变量

import vuex from "vuex"

vue.use(vuex)

    state :localStorage.getItem('state') ? JSON.parse(localStorage.getItem('state')) 
    // state 
     : {
        //  通讯录
        linkman:[],
        selfmes:""
      },//基本数据存放
     
    getters:{
        // usuallyLists(state){
        //     return state.usuallyList
        // },
    },
    mutations:{
        getlinkman(state,value){
            state.linkman=value;
        },
        getmes(state,value){
            state.selfmes=value;
        },
        getphoto(state,value){
            state.selfmes.photo=value
        }

 
    },
    actions:{
        switch_d(context){//这里的context和我们使用的$store拥有相同的对象和方法
           
            //你还可以在这里触发其他的mutations方法
        },
    }
})

# 在main.js里引用

import Vuex from 'vuex'

})
